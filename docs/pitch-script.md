# Pitch Script: AI Resume Analyzer Agent

## Problem
Recruiters and students waste a lot of time manually checking resumes for skills, gaps and job fit.

## Solution
An AI Agent that:
- Reads a resume PDF
- Uses Gemini to understand skills, experience and intent
- Maps skills to suitable roles
- Highlights gaps and gives concrete improvement suggestions
- Provides an ATS-style score and feedback

## Demo Flow (for video)
1. Show the web UI with futuristic theme.
2. Upload a sample resume.
3. Show the AI dashboard with:
   - Summary
   - Skills extracted
   - Strengths & improvements
   - Suggested job roles
4. Explain multi-agent design:
   - Resume Agent (Gemini)
   - Skill Gap Agent
   - Job Match Agent
   - Memory Service (session-based)
5. Conclude with how this saves time and improves decisions.
